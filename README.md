# AWS EC2 Docker Deployment

## Project Overview

This project demonstrates the deployment of a containerized web application on AWS EC2 using Ubuntu Linux, Docker, and Nginx.

The objective was to gain hands-on experience with:

* AWS EC2 provisioning
* Linux server administration
* Docker installation and management
* Container image creation
* Networking and security configuration
* Remote administration using SSH

---

## Project Workflow

1. Launch AWS EC2 Ubuntu instance
2. Configure Security Groups
3. Connect using SSH
4. Install Docker
5. Create Dockerfile
6. Build Docker image
7. Run container
8. Verify application through public IP
